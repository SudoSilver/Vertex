# Installing flare
*NOTE:In document you will find how to build flare from source. If you dont wanna do it just use prebuilded binaries at the github repo althrought they are just for Linux*

---

You will use install.py skript in the root of this project
## Dependecies
- Python 3 or greater
- Rust program language
- git
## Installation process
First clone the repo and cd in to it
```bash
git clone github.com/DomioKing653/Flare
cd Flare
```

now run the install.py skript:
```bash
python install.py
```

and the final build should be in ./targe/release/

# Congrats you have just build flare from source
